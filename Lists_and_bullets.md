# Lists and bullet points

To create lists, begin a line with an asterisk, or minus:

 * item 1
 * item 2
 * item 3
 
 This is with minus:
 
 - item 1 
 - item 2
 - item 3
 
 You can make sub points with two spaces before a point:
 
 * item 1
   * sub-item 1
   * sub-item 2
* item 2

If you want to create a numbered listL

1. item 1
1. item 2
1. item 3
